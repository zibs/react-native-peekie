# Contributing

Thanks for your interest in contributing to **peekie**!

## Getting started

1. Fork the repo and create a feature branch.
2. Install dependencies in the module and example app:
   - `cd module && npm install`
   - `cd module/example && npm install`
3. Run the example app (requires a development build):
   - `cd module/example && npx expo start`

## Development workflow

- Keep changes focused and add tests or demo updates when applicable.
- Run `npx expo-module typecheck` from `module/` before opening a PR.
- Update `module/README.md` if the public API changes.

## Submitting a PR

- Describe the problem and the solution.
- Include screenshots or short videos for UI changes.
- Link to relevant issues.

## Code of Conduct

By participating, you agree to follow the [Code of Conduct](CODE_OF_CONDUCT.md).
